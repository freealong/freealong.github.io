新建openvpn

Presuming you’ve already installed OpenVPN, if not use this command
sudo apt-get install openvpn network-manager-openvpn
Click Network Connections icon -> VPN Connections -> Configure VPN, then Add -> Import a saved VPN connection (in the scroll down menu), then Create, now select your downloaded and chopped OpenVPN config file, xxx.ovpn. In the new Editing vpngate_xxx window with Connection name and Gateway already filled-in, change the Authentication mode from Certificates (TLS)
to Password. Fill-in as follows:

editing_vpn
username: vpn
password: vpn
CA Certificate: ca.crt file
然后在点击高级按钮，选择自定义端口，设为配置文件名中的端口; 如果是TCP则勾选 Use a TCP connection选项
Now save and Close Network Connections.
